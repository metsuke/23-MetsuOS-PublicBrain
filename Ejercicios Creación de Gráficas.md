# Ejercicios Creación de Gráficas

[[Aprender sobre Matematicas]]

## y = 4x + 1

Dibujamos $$y = 4x + 1$$

```functionplot
---
title: y = 4x + 1
xLabel: x
yLabel: y
bounds: [-10,10,-10,10]
disableZoom: false
grid: true
---
y = 4x + 1
```
