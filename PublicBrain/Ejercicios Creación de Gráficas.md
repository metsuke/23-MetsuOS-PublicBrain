## y = 4x + 1

Dibujamos $$y = 4x + 1$$

```functionplot
---
title: y = 4x + 1
xLabel: x
yLabel: y
bounds: [-10,10,-10,10]
disableZoom: false
grid: true
---
y = 4x + 1
```

## 2x - y = -1

Despejamos y y dibujamos $$2x - y = -1 => -y = -1 - 2x =>  -1*-y = -1(-1 - 2x) => y = 1 + 2x$$

```functionplot
---
title: y = 1 + 2x
xLabel: x
yLabel: y
bounds: [-10,10,-10,10]
disableZoom: false
grid: true
---
y = 1 + 2x
```
